#  AIAD 智绘设计 施工图设计说明编制助手

基于建筑设计单位的历史经验和技术沉淀，在施工图设计阶段，基于LLM为设计师提供快速的半自动设计说明文字生成功能。同时，提供AI校对、检查。
产品形态：
遵循设计师工作习惯，参考Auto CAD等设计软件，在一个页面呈现所有功能并完成人机互动的操作。包括不同知识库的权重选择排序、项目实例信息的人工输入、分章节分段的生成+prompt改写后重新生成+人工修改。 
所有工作过程中，不同章节内容会在不同时间进行，AI生成、人工修改、人工校对、定稿、再次修改等需要有明确的状态标识，并可查阅人机互动的记录。
不同章节内容AI生成的prompt会提供预制版本，但同时支持每次生成的人工调整。prompt以结构化的形式提供可编辑条件，如任务（生成的具体章节内容）、通用要求（准确、专业、文风等）、特殊要求（项目特征、甲方要求、修改意见等）。专业场景下prompt字数多。
AI会调用后台的RAG知识库，针对具体生成的内容，应显示所参考的内容出处以及每个章节具体生成时AI的额外考虑，方便用户采信与调整。

## ✨ 功能特性

### 🤖 AI 智能生成
- **智能文档生成**：输入项目基本信息，AI 自动生成符合规范的设计说明初稿
- **AI 助手**：提供续写、改写、优化等智能辅助编辑功能
- **章节管理**：支持章节化编辑，所见即所得

### 📚 知识库集成
- **规范条文检索**：集成 500+ 建筑规范条文，一键插入标准表述
- **分类管理**：按防火、结构、机电等专业分类管理
- **智能搜索**：快速定位相关规范内容

### ✍️ 可视化编辑
- **Markdown 编辑器**：支持富文本编辑，实时预览
- **拖拽排序**：大纲章节支持拖拽重新排序
- **实时保存**：自动保存，防止数据丢失

### 📄 多格式导出
- **PDF 导出**：生成标准格式的 PDF 文档
- **Markdown 导出**：导出为 Markdown 格式
- **AutoCAD 图框**：支持导出到 AutoCAD 图框格式

### 🔄 版本管理
- **历史版本**：自动保存历史版本，随时回溯修改记录
- **变更追踪**：记录文档修改历史

## 📖 使用指南

### 1. 创建新文档

1. 在首页点击"新建文档"按钮
2. 填写项目基本信息（标题、建筑类别、结构类型等）
3. 选择是否自动生成项目概要
4. 设置文档大纲结构
5. 确认信息后生成文档

### 2. 编辑文档

1. **左侧大纲**：点击章节快速跳转，支持拖拽排序
2. **中间编辑区**：直接编辑文档内容，支持富文本编辑
3. **右侧AI助手**：使用AI功能优化内容（开发中）

### 3. 知识库使用

1. 切换到"知识库"标签页
2. 按分类浏览规范条文
3. 搜索相关规范内容
4. 点击"插入"按钮将内容添加到文档

### 4. 导出文档

1. 点击顶部工具栏的"导出"按钮
2. 选择导出格式（PDF/Markdown/AutoCAD）
3. 等待生成完成

## 🎯 核心功能详解

### 文档向导

文档向导提供分步骤的文档创建流程：

1. **标题设置**：输入文档标题和基本配置
2. **项目概要**：自动生成或手动编写项目概要
3. **大纲设置**：自定义文档章节结构
4. **内容生成**：预览并生成最终文档

### 智能编辑器

- **实时编辑**：支持所见即所得的编辑体验
- **章节管理**：左侧大纲与中间内容区域联动
- **自动保存**：每30秒自动保存一次
- **版本控制**：支持查看历史版本

### AI 助手功能

- **内容续写**：基于现有内容智能续写
- **内容改写**：优化现有表述
- **规范检查**：检查内容是否符合相关规范
- **智能建议**：提供编辑建议和优化方案

## 🔧 开发说明

### 组件说明

- **Editor.vue**：主编辑器组件，包含完整的编辑功能
- **DocumentWizard.vue**：文档创建向导
- **Home.vue**：应用首页，展示项目列表

### 数据流

1. 项目数据存储在 `localStorage` 中
2. 编辑器状态通过 Vue 3 Composition API 管理
3. 路由参数传递项目ID

```css
  --gray-100: #f3f4f6;
  /* ... 更多颜色变量 */
}
```
